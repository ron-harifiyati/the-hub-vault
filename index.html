The Foundation

What is GitHub?

Forget the technical jargon. Here is what GitHub actually does for you.

📁

The Folder

GitHub is a cloud-based home for your project files. Think of it as Google Drive, but built specifically for creators who want to show their work professionally.

Every file you upload lives there permanently — accessible from any device, anywhere in Zimbabwe.

⏳

The Time Machine

GitHub remembers every version of your work. If you accidentally delete a slide on your iPad today, GitHub has the version from last week. Nothing is ever truly lost.

This is called version history — and it is one of the most powerful tools a professional can have.

🛂

The Digital Passport

Your GitHub profile is how the Matter Career Readiness Institute (MCRI) and hiring companies see who you are before they ever meet you.

A filled GitHub profile with real projects says: "I am serious. I do the work. Here is the proof."

How We Use It

Three Reasons GitHub Matters Here

At every innovation hub across Zimbabwe, we use GitHub for the same three things.

📄

Reports

Facilitators keep sponsors updated with weekly PDF reports — uploaded directly to the hub's GitHub repository. One link. All the evidence. No email attachments.

🏆

Portfolios

Students build a track record of commits that proves they are ready for the Institute. Your GitHub is your portfolio — and it speaks for itself.

👀

Transparency

Sponsors, hire-ups, and the MCRI can watch the birth of a developer ecosystem in real-time — no login required. The work is public. The growth is visible.

The Big Idea

"Your work is too valuable to lose.
Stop emailing. Start committing."

iPads get wiped at the end of every cohort. Emails get buried. Hard drives fail. GitHub does not. The hub that builds on GitHub builds a legacy — not just a workshop.
